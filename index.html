Image Super-Resolution

Developed an advanced image super-resolution model to significantly enhance the quality of low-resolution images. Utilized the DIV2K dataset to train the model, leveraging deep learning techniques to upscale images with high fidelity.

Technologies Used: Python, PyTorch, OpenCV

Attained a Peak Signal-to-Noise Ratio (PSNR) of 30.5 dB and a Structural Similarity Index (SSIM) score of 0.85. These metrics reflect substantial improvements in image clarity and detail preservation, making the enhanced images suitable for professional applications.

Speech Emotion Recognition

Engineered a model to accurately identify emotions from speech recordings using the RAVDESS dataset. This project involved extracting features with Mel-Frequency Cepstral Coefficients (MFCC), training a classification model, and evaluating its performance on emotion recognition tasks.

Technologies Used: Python, TensorFlow, Librosa

Achieved an impressive accuracy rate of 82.58%, demonstrating the model's efficacy in distinguishing between different emotional states from speech. This system can be applied to enhance user experience in various applications such as virtual assistants and customer service.
